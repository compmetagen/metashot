Homepage: https://drep.readthedocs.io/en/latest/index.html

## Contents

  * drep v2.2.3
  * ANIcalculator (aka gANI) v1

  * pplacer v2.1
  * HMMER v3.2.1
  * Prodigal v2.6.3
  * MUMmer v3.23
  * Mash v2.1
